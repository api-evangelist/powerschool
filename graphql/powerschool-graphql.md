# PowerSchool GraphQL API

PowerSchool provides K-12 student information systems. The API covers students, enrollments, courses, grades, attendance records, assignments, teachers, sections, parents, and district reporting for SIS integration.

**Endpoint:** No native GraphQL endpoint — conceptual schema derived from PowerSchool REST API
**Documentation:** https://support.powerschool.com/developer/
